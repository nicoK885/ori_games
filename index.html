Neue Version ohne Index-Anpassung

  1. Lege im Repo einen neuen Ordner an, z. B. many_oris_v3.
  2. Packe dort deine Spiel-Datei als index.html hinein.
  3. Committe die Änderung auf main.
  4. Nach dem GitHub-Pages-Refresh erscheint die Version automatisch hier.

Optional: bessere Karteninfos

Lege im Spielordner zusätzlich game.json an:

{ "title": "Many Oris V3", "description": "Neue Version mit mehr Oris und Effekten.", "icon": "✨", "tag": "Neu", "series": "many-oris", "priority": 300 }
